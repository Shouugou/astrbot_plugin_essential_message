# Changelog

## v0.3.0 (2026-05-12)

### 变更

- 移除配置中的平台适配器选项，平台固定为 `aiocqhttp`。
- 重构订阅群聊配置，支持在插件配置页为每个群设置发送条数和发送时间。

### 修复

- 修复卡片内容过高时图片被裁切的问题。
- 修复卡片竖直方向被裁切的问题。

## v0.2.1 (2026-05-12)

### 修复

- 修复卡片内容过高时图片被裁切的问题，改用 `full_page: True` 自适应内容高度。
- 修复卡片竖直方向被裁切的问题，移除 flex 布局改用流式排列。

## v0.2.0 (2026-05-12)

### 新增

- 支持在精华消息中展示图片内容。
- 支持通过 AstrBot 配置页管理订阅群聊。

## v0.1.0 (2026-05-04)

### 新增

- 初始版本，实现每日定时发送 QQ 群精华消息。
- 群精华消息渲染为带头像、昵称、加精信息的图片卡片。
- 支持群内指令管理订阅：`/精华开启`、`/精华关闭`、`/精华条数`、`/精华时间`、`/群精华`、`/精华状态`。
- 支持每个群独立设置每日发送时间和每次发送条数。
- 订阅数据持久化到 AstrBot KV 存储和本地 JSON 文件。
- 当天内自动去重，避免重复发送同一条精华。
