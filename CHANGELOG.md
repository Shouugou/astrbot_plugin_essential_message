# 更新日志

## v1.0.3 (2026-05-14)

### 修复

- 修复 `/群精华` 指令可能错误创建订阅的问题，现在只有 `/精华开启` 指令才能创建订阅。

## v1.0.2 (2026-05-14)

### 修复

- 修复 `_stop_llm` 方法逻辑错误导致 `/群精华` 指令可能重复发送的问题。

## v1.0.1 (2026-05-14)

### 修复

- 修复调用 OneBot `get_essence_msg_list` 接口时参数格式错误导致发送群精华失败的问题。

## v1.0.0 (2026-05-13)

### 变更

- 使用 `template_list` 重写订阅群聊配置，支持在配置页为每个群设置群号、发送条数和发送时间。
- 移除配置中的平台适配器选项，平台固定为 `aiocqhttp`。
- 订阅数据仅保存到本地文件，移除 KV 存储依赖。
- `/精华关闭` 改为直接删除订阅。
- 已发送消息 ID 持久化到本地文件，跨重启保持当日去重。
- 插件卸载时自动清理本地数据文件。

### 修复

- 修复卡片内容过高和竖直方向被裁切的问题。
- 修复开启订阅后立即发送群精华的问题。
- 修复仅剩一条订阅时无法删除的问题。
- 每日精华去重，全部发送完后返回提示。

## v0.3.1 (2026-05-12)

### 变更

- 订阅数据仅保存到本地文件，移除 KV 存储依赖。
- `/精华关闭` 改为直接删除订阅，而非仅标记为禁用。
- 插件卸载时自动清理本地订阅文件，重装不会残留旧配置。

### 修复

- 修复仅剩一条订阅时无法删除的问题。

## v0.3.0 (2026-05-12)

### 变更

- 使用 `template_list` 重写订阅群聊配置，支持在插件配置页为每个群设置群号、发送条数和发送时间。
- 移除配置中的平台适配器选项，平台固定为 `aiocqhttp`。
- 移除配置中的订阅群聊概览字段。

## v0.2.1 (2026-05-12)

### 修复

- 修复卡片内容过高时图片被裁切的问题，改用 `full_page: True` 自适应内容高度。
- 修复卡片竖直方向被裁切的问题，移除 flex 布局改用流式排列。

## v0.2.0 (2026-05-12)

### 新增

- 支持在精华消息中展示图片内容。
- 支持通过 AstrBot 配置页管理订阅群聊。

## v0.1.0 (2026-05-04)

### 新增

- 初始版本，实现每日定时发送 QQ 群精华消息。
- 群精华消息渲染为带头像、昵称、加精信息的图片卡片。
- 支持群内指令管理订阅：`/精华开启`、`/精华关闭`、`/精华条数`、`/精华时间`、`/群精华`、`/精华状态`。
- 支持每个群独立设置每日发送时间和每次发送条数。
- 订阅数据持久化到 AstrBot KV 存储和本地 JSON 文件。
- 当天内自动去重，避免重复发送同一条精华。
